# WeLearn 自动学习工具

## 项目简介

本人是一位来自黑大的苦逼学生，因不满校内各种付费代刷课，所以制作了这款软件。

这是一个基于Python和PyQt5开发的WeLearn平台自动学习工具，支持自动完成课程作业和刷学习时长功能。

通过直接发送提交包完成每节课的学习，后续还会写自动刷知到、学习通等软件。

## 功能特点

### 🎯 刷作业模式
- ✅ 自动完成课程作业
- ✅ 支持固定正确率和随机正确率
- ✅ 可指定单元或全部单元自动完成
- ✅ 实时显示学习进度和统计信息

### ⏰ 刷时长模式  
- ✅ 自动增加学习时长
- ✅ 支持固定时长和随机时长
- ✅ 发送心跳包模拟真实学习行为
